# YSN — Young Scholar Network

Official website for the **NLP@U Young Scholar Network** at Utrecht University.

🌐 Live site: [https://nlpysn.github.io/](https://nlpysn.github.io/)

## About

YSN is a community of PhD candidates, postdocs, and early-career researchers in Natural Language Processing and related fields at Utrecht University and UMCU. We organize talks, workshops, and networking events.

## Run Locally

```bash
python -m http.server 8000
```

Then open [http://localhost:8000](http://localhost:8000).
